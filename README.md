#  🌐 Personal Portfolio Website

## 🚀 Overview

This is my personal portfolio website built using React. It showcases my projects, technical skills, and background as an aspiring software developer. The goal of this project is to present my work in a clean, responsive, and professional format.

---

## ✨ Features

* Responsive design (mobile-friendly)
* Project showcase with GitHub links
* Clean and modern user interface
* Contact section for easy communication
* Fast performance using Vite

---

## 🛠️ Tech Stack

* React
* Vite
* JavaScript
* HTML5
* CSS3

## 🌍 Live Demo

https://f-anselm88.github.io/my-portfolio/

---

## ⚙️ Installation & Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/f-anselm88/my-portfolio.git

# Navigate into the project folder
cd my-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
